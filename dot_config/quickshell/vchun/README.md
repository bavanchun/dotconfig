# Quickshell Vchun

Primary Quickshell config for Hyprland.

Start it manually:

```bash
bash ~/.config/hypr/scripts/quickshell-monitor.sh start
```

Useful commands:

```bash
bash ~/.config/hypr/scripts/quickshell-monitor.sh status
bash ~/.config/hypr/scripts/quickshell-monitor.sh restart
bash ~/.config/hypr/scripts/quickshell-monitor.sh stop
bash ~/.config/hypr/scripts/quickshell-monitor.sh foreground
```

This config started as a small top bar and now replaces the Waybar autostart.
The old Waybar files are still kept in chezmoi as a rollback path.
