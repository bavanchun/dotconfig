# Quickshell Codex Safe

Trial Quickshell config for Hyprland.

Run it without replacing Waybar:

```bash
bash ~/.config/hypr/scripts/quickshell-test.sh start
```

Useful commands:

```bash
bash ~/.config/hypr/scripts/quickshell-test.sh status
bash ~/.config/hypr/scripts/quickshell-test.sh restart
bash ~/.config/hypr/scripts/quickshell-test.sh stop
bash ~/.config/hypr/scripts/quickshell-test.sh foreground
```

This config intentionally starts as a small top bar so the Quickshell stack can
be tested safely before replacing the current Waybar monitor.
