## Neovim  (Leader = Space)

### Registers & Clipboard

| Keybind | Action |
|---|---|
| `x` | Delete char (no register) |
| `Leader + p / P` | Paste from register 0 |
| `Leader + c / C` | Change (no register) |
| `Leader + d / D` | Delete (no register) |
| `dw` | Delete word backwards |

### Tabs & Windows

| Keybind | Action |
|---|---|
| `te` | New tab |
| `Tab` / `Shift+Tab` | Next/Prev tab |
| `ss` | Horizontal split |
| `sv` | Vertical split |
| `sh/sk/sj/sl` | Move between windows |
| `Ctrl+w + ←/→/↑/↓` | Resize window |

### Navigation & Editing

| Keybind | Action |
|---|---|
| `+` / `-` | Increment / Decrement |
| `Ctrl + m` | Jump forward (jumplist) |
| `Ctrl + j` | Next diagnostic |
| `Leader + o / O` | New line below/above |
| `Leader + r` | Convert hex → HSL |
| `Leader + i` | Toggle inlay hints |
